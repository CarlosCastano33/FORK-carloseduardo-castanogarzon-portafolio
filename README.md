# 🚀 Portfolio & Resume Website

Este proyecto es una página web personal diseñada para funcionar como hoja de vida digital y portafolio profesional.  
El objetivo principal es presentar información sobre experiencia, habilidades, proyectos y tecnologías utilizadas de una forma moderna, interactiva y responsive.

## 🌐 Despliegue

La aplicación está desplegada en Vercel y puede visualizarse aquí:

👉 https://fork-carloseduardo-castanogarzon-po-five.vercel.app/

## ✨ Características

- Diseño moderno y responsive
- Presentación profesional tipo CV
- Sección de proyectos destacados
- Tecnologías y habilidades técnicas
- Navegación fluida e interactiva
- Optimización para diferentes dispositivos

## 🛠️ Tecnologías utilizadas

- React
- Next.js
- TypeScript
- Tailwind CSS
- Vercel

## 📦 Instalación y ejecución local

Clona el repositorio:

```bash
git clone https://github.com/202601-Ingenieria-Web/carloseduardo-castanogarzon-portafolio.git
```

Ingresa a la carpeta del proyecto:

```bash
cd carloseduardo-castanogarzon-portafolio
```

Instala las dependencias:

```bash
npm install
```

Ejecuta el servidor de desarrollo:

```bash
npm run dev
```

La aplicación estará disponible en:

```bash
http://localhost:3000
```

## 🎯 Propósito del proyecto

Este proyecto fue desarrollado con el fin de:

- Mostrar proyectos y experiencia profesional
- Centralizar información personal y técnica
- Servir como carta de presentación digital
